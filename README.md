# **Curso-Git**

### **Trabajo con Ramas**



**-----------------------------------------------------------------------------------------**



* ##### **Objetivo**



Aprender a usar Git creando ramas y subiendo cambios individuales.



**-----------------------------------------------------------------------------------------**



* ##### **Instrucciones**



1. Clona el repositorio:

git clone URL\_DEL\_REPOSITORIO



2\. Entra al proyecto:

cd actividad-git-clase



3\. Crea tu propia rama con tu nombre:

git checkout -b rama-tuNombre



* *Ejemplo:*



git checkout -b rama-juan



**-----------------------------------------------------------------------------------------**



* ##### **Actividad**



Dentro de la carpeta /alumnos/ crea un archivo con tu nombre:



*alumnos/tuNombre.txt*



Contenido del archivo:

* Nombre completo
* Edad
* Carrera
* Pasatiempo favorito
* Algo interesante sobre ti



**-----------------------------------------------------------------------------------------**



* ##### **Guardar cambios**



git add .

git commit -m "Agrego mi archivo personal"

git push origin rama-tuNombre



**-----------------------------------------------------------------------------------------**



* ##### **Reglas**



❌ No trabajar en la rama main

❌ No modificar archivos de otros compañeros

✅ Usar tu propia rama

✅ Subir tus cambios
